# feedMastodon

Feed Mastodon with RSS Feeds

## Requirements

* Python 3
* Mastodon API
* feedparser

## Installation and usage

```
git clone git@framagit.org:thomasb22/feedMastodon.git
cd feedMastodon
feedMastodon.py
```
